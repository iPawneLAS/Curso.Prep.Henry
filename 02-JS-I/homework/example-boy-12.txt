Variables: Una variable es un cajón virtual donde nosotros podemos guardar algo para usarlo más tarde. Nosotros podemos designar cuando la creamos, que tipo de cosas va a guardar dicho cajón.
Strings: Un string es (Tomando de referencia el ejemplo anterior) una etiqueta (Tipo de dato) que le vamos a poner al cajón, para que ahí dentro solo se guarde TEXTO. Siempre se va a distinguir si es string a todo lo que esté dentro de unas comillas dobles o simples ("",'').
Funciones: Una función es un cajón, donde se van a guardar instrucciones creadas por nosotros o alguien más, para usarla cuando se necesite, una función es un objeto llamable, por lo que se lo puede llamar en alguna parte del codigo. Argumentos: Son acciones puestas adentro de las funciones para que cumpla con lo que especificamos, return, por ejemplo es una función que devuelve el resultado de una operación dentro de nuesta función.
Declaraciones IF: IF es un condicional, un condicional es una desición que se va a tomar en base a ciertas circunstancias, IF significa SI, cuando este se usa para comparar algo. EXAMPLE: SI (IF) Yo tengo hambre - Ir a comer.
Valores booleanos: Hace referencia al código binario, y tiene solo dos resultados posibles, verdadero o falso, y pueden usarse para argumentar en alguna función o en alguna declaración también.
